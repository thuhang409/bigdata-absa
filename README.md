# Big Data Project - UIT
 
In this course project, we build a big data application that analyzes customer reviews for a restaurant in terms of aspects (ABSA). 
- First, we created a new dataset that crawled from the popular Vietnamese food reviews, we has the kappa score up to 84.52%. 
- Second, we used PySpark, BigDL and Zoo-Analysis to build the pipeline for this task. 
- Finally, we deployment it using streamlit.
